If the cursor is left idle for 2000 milliseconds, the cursor will disappear. When the cursor is active again, it will return back to it's default state.
This system active all through the 'HTML' element.

JS Code snippet:
$(document).ready(function() {
var dummy;
	$(document).mousemove(function() {
		clearTimeout(dummy);
			$('html').css({cursor: 'auto'});
		dummy = setTimeout('hide();', 2000);
	});
});

function hide() {
	$('html').css({cursor: 'none'});
}
