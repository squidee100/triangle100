Chat Commands

A list of commands that can be used in squidee_'s twitch chat.

squideebot_ commands

!ping returns a generic message from the bot
!commands returns a list of commands
!roll or !dice returns a random number from 1 to 6
!followage returns the amount of time the user has followed squidee_

Nightbot commands

!game returns squidee_'s current game
!title returns the stream title
